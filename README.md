# Sorting algorithms

## Os argumentos devem ser passados na função main
#### O primeiro argumento é um número inteiro, que representa o algoritmo
#### O segundo argumento é o diretório do arquivo

```
    ./sorting 6  instancias-num/num.1000.1.in
```

Alg. | Nome
-----| ------
 1   | CountingSort
 2   | RadixSort
 3   | SelectionSort
 4   | InsertionSort
 5   | BubbleSort
 6   | MergeSort
 7   | QuickSort
